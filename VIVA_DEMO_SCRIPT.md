Use **Start demo walkthrough** on the home page before presenting.

## 1. Introduction (30 sec)
> "I built Finance Analytics Hub — a unified investment planning platform for mutual funds, crypto, and goals, with live market data and Monte Carlo simulation."

Open **Project Overview** → show student card, objectives, architecture.

## 2. Command Center (1 min)
- Point to **Nifty 50 / Sensex** (live India data)
- Show **Crypto Mood** gauge (fear/greed style)
- Run **Monte Carlo** — drag volatility, explain percentile bands

## 3. Mutual Funds (1 min)
- Generate plan with demo profile
- Show equity-debt donut + **retirement projection chart**
- Download JSON plan

## 4. Goal Planner (1 min)
- Show 3 goals with required SIP vs allocated SIP
- Highlight **Success probability** (Monte Carlo)
- Show timeline chart

## 5. AI Advisor (1 min)
- Show health score + mood + prioritized actions
- **Download HTML report** — open in browser to impress

## 6. Closing (30 sec)
> "Future scope includes ML risk profiling and NSE equity integration. Thank you."
